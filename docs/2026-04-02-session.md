## 7. LLMBackend 抽象层（P0）

### 新文件
- `core/llm_backend.py` — 3 个 backend class:
  - `LocalVLLM`: vLLM 本地推理（Qwen3-14B-AWQ）
  - `AnthropicBackend`: Claude API（消息格式转换 + 工具格式转换）
  - `OpenAIBackend`: OpenAI 兼容 API（GPT-4, DeepSeek 等）
- `config.py` — `LLM_PROVIDER` / `LLM_API_KEY` / `LLM_MODEL` 环境变量

### 统一接口
```python
backend = get_backend("anthropic", api_key="sk-...")
response = await backend.chat(messages, tools=tool_defs)
tool_calls = backend.format_tool_calls(response)
```
