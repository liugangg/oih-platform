## 关键陷阱（血泪经验）
1. IgFold 只适用于抗体序列 — de novo binder 的 pLDDT 无意义（0.4-12.6）
2. FreeSASA 不接受 CIF — 需先用 BioPython MMCIFParser 转 PDB
3. RDKit EmbedMolecule 对大分子 ADC 失败 — 需 2D SMILES fallback
4. .get() 当 key 存在但 value=None 时返回 None（不是 default）— 用 `or 0`
5. linker_library 没有 NHS-amine 类型 — 需补充
6. 服务重启会丢失 running pipeline 状态 — AF3 子任务在容器内继续但编排丢失
