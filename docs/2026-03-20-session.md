### 1. pocket_guided_binder_pipeline 重构 — 多维口袋评分（11步→15步）

**旧流程**：P2Rank top pocket → 取 top 6 残基 → DiffDock 盲对接 → RFdiffusion
**新流程**：5维评分 + Qwen 结构生物学家选择 → DiffDock 成药性参考 → RFdiffusion + BindCraft 并行

**评分公式**：
```
composite = p2rank_prob × 0.2 + sasa × 0.2 + conservation × 0.2 + rag × 0.3 + electrostatics × 0.1
```

| 维度 | 方法 | 权重 |
|------|------|------|
| P2Rank | ML probability | 0.2 |
| SASA | FreeSASA mean_residue/150, cap 1.0 | 0.2 |
| Conservation | 1 - mean_normalised_bfactor | 0.2 |
| RAG | 文献残基重叠度 0/0.5/1.0 | 0.3 |
| Electrostatics | 带电残基(K/R/E/D/H)比例 | 0.1 |

**15步流程**：
1. fetch_pdb
2. RAG 文献检索（直接调用 HybridRetriever，不走 HTTP）
3. fpocket + P2Rank（并行，top 5）
4. FreeSASA per pocket（表面暴露度）
5. B-factor conservation + electrostatics
6. 复合评分 + Qwen 口袋选择（enable_thinking=False）
7. DiffDock 成药性参考（仅对选中口袋）
8a. RFdiffusion binder backbone 设计
8b. BindCraft 并行（asyncio.gather，失败不阻塞）
9. ProteinMPNN 序列设计
10. 合并 MPNN + BindCraft 候选（BindCraft 最多 2 条入池）
11. AF3 验证（top 5，ipTM≥0.6）
12-15. ADC 构建（FreeSASA → Linker → MMAE → RDKit DAR=4）

**新增 9 个 helper 函数**：`_parse_p2rank_residues`, `_compute_bfactor_conservation`, `_compute_electrostatics_from_pdb`, `_compute_sasa_score_for_pocket`, `_compute_rag_score`, `_extract_residue_numbers_from_text`, `_rag_search_pocket_context`, `_compute_freesasa_per_residue`, `_qwen_select_pocket`

### 2. 三个运行时 Bug 修复

**Bug 1 — RAG self-HTTP 死锁**
- 症状：pipeline 挂死在 RAG 步骤（单 worker uvicorn）

**Bug 2 — Qwen3 thinking mode 空 content**
- 症状：Qwen selection 静默失败，错误信息为空字符串
